排班流程

1. 選月份與櫃位
店長先選要排的月份，再切換櫃位 A / B / 快閃櫃。
開始
2. 確認兩班制
預設早班 10:00-16:00、晚班 16:00-22:00，可作為固定班別模板。
模板
3. 指派員工
在月曆每日早/晚班格子指定員工；一格代表一個班次。
排班
4. 檢查衝突
系統提示同員工撞班、超時、櫃位缺班或未排滿。
檢查
5. 發布班表
發布後員工只能在「我的班表」查看自己的班次。
發布
6. 月底薪資試算
用排班時數 x 時薪，加上每日個人業績抽成。
薪資
